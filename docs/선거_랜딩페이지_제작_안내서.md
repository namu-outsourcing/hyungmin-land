# 선거 홍보 랜딩페이지 제작 프로젝트 안내 및 요청사항 확인서

**작성일:** 2026년 3월

## 1. 프로젝트 개요

이번 작업은 대학교 선거 후보자의 공약 및 정보를 알리는 단순 정보 전달용 랜딩페이지 제작입니다. 모바일과 PC 모두에서 깔끔하게 보이도록 제작하며, 선거 기간에 카카오톡 링크 공유 및 QR코드 등으로 배포할 수 있게 제공합니다.

## 2. 작업 범위

아래 표를 기준으로 포함 항목과 미포함 항목을 확인해 주세요.

| ✅ 포함 항목                 | ❌ 미포함 항목                    |
| :--------------------------- | :-------------------------------- |
| • 랜딩페이지 1개 제작        | • 로고 / 이미지 직접 제작         |
| • 모바일/PC 반응형 지원      | • 도메인 구매 및 연장 (별도 실비) |
| • 콘텐츠 입력 (자료 제공 시) | • 선거 이후 유지보수              |

> [!NOTE]
> 도메인이 필요하신 경우 별도 실비(연간 약 1~2만원)가 발생할 수 있습니다. 무료 링크(공개 주소)로도 충분히 공유 가능합니다.

## 3. 비용 및 결제

합의된 금액 기준으로 선금/잔금 구조로 진행합니다.

| 항목          | 내용                            |
| :------------ | :------------------------------ |
| 총 제작 비용  | 200,000원                       |
| 선금 (착수금) | 100,000원 (작업 시작 전)        |
| 잔금          | 100,000원 (최종 납품 후)        |
| 수정 횟수     | 2회 포함 / 추가 수정 시 1~2만원 |

## 4. 진행 순서

요구사항 파악 → 선금 50% 입금 → 시안 공유 & 수정 → 최종 납품 + 잔금 50%

> [!IMPORTANT]
> 자료(글·사진)를 전달해 주시는 시점부터 실질적인 작업이 시작됩니다. 자료가 늦어질 경우 납기일도 그만큼 미뤄질 수 있습니다.

## 5. 안내사항

- 수정은 2회까지 포함되며, 추가 수정 발생 시 건당 1~2만원이 협의됩니다.
- 자료 제공이 늦어질 경우 납기일이 조정될 수 있으며, 이 점 미리 양해 부탁드립니다.
- 소통은 카카오톡으로 진행하며, 시안 공유 및 피드백도 카카오톡을 통해 진행합니다.
